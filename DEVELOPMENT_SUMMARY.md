# 모두의수선 개발 현황 및 다음 단계

## 📅 작성일: 2024년 11월

---

## ✅ 완료된 기능

### 1. 인증 시스템 (Auth)
- ✅ **스플래시 페이지** - 앱 시작 화면
- ✅ **로그인 페이지** - 이메일/비밀번호 로그인, 소셜 로그인 UI (Google, Naver, Kakao)
- ✅ **회원가입 페이지** - 이메일/전화번호 중복 체크, 약관 동의, 소셜 회원가입 UI
- ✅ **비밀번호 찾기 페이지** - 이메일 기반 비밀번호 재설정 링크 발송
- ✅ **이용약관 페이지** - 서비스 이용약관 상세 내용
- ✅ **개인정보처리방침 페이지** - 개인정보 처리방침 상세 내용

### 2. 주문 관리 (Orders)
- ✅ **홈 페이지** - 배너, 최근 주문 미리보기, 수선 신청 플로우 시작
- ✅ **주문 목록 페이지** - 상태별 필터링, 날짜 필터링, 페이징 (10개/페이지)
- ✅ **주문 상세 페이지** - 상태별 UI, 타임라인, 사진 편집, 핀 관리, 수거 취소 (BOOKED 상태)
- ✅ **수선 신청 플로우**:
  - 의류 종류 선택
  - 사진 촬영/선택
  - 이미지 핀 편집 (최대 5개, 드래그 이동, 메모 입력)
  - 수선 타입 선택
  - 수선 부위 선택
  - 상세 치수 입력
  - 최종 확인
  - 수거 신청
  - 결제
- ✅ **이미지 핀 편집**:
  - 핀 추가 (탭)
  - 핀 드래그 이동 (넓은 드래그 영역)
  - 핀 메모 입력/수정 (핀 클릭 시)
  - 핀 삭제 (메모 입력창에서)
  - 최대 5개 제한

### 3. 프로필 관리 (Profile)
- ✅ **마이페이지** - 사용자 정보, 메뉴 리스트
- ✅ **회원정보** - 이름, 전화번호, 이메일 수정, 프로필 이미지
- ✅ **비밀번호 변경** - 현재 비밀번호 확인, 새 비밀번호 설정
- ✅ **배송지 설정** - 배송지 목록, 추가, 수정, 삭제
- ✅ **결제수단 관리** - 결제수단 목록 (UI)
- ✅ **결제내역** - 결제 목록, 날짜 필터링, 페이징, 영수증 보기
- ✅ **포인트 내역** - 포인트 적립/사용 내역, 날짜 필터링, 페이징
- ✅ **친구초대** - 초대 링크 공유 (UI)
- ✅ **공지사항** - 공지사항 목록, 페이징
- ✅ **고객센터** - 고객센터 정보 (UI)
- ✅ **앱 설정** - 앱 설정 메뉴 (UI)

### 4. 공통 기능
- ✅ **사업자 정보 푸터** - 아코디언 방식, 모든 페이지에 적용
  - 회사명, 대표자, 사업자등록번호, 통신판매업신고번호
  - 주소, 개인정보관리책임자, 이메일, 고객센터
  - 이용약관, 개인정보처리방침 링크
- ✅ **라우팅** - GoRouter 기반 네비게이션
- ✅ **테마** - 일관된 디자인 시스템

### 5. 관리자 페이지 (Admin)
- ✅ **기본 구조** - Next.js 기반 관리자 콘솔
- ✅ **대시보드** - 운영 통계, 최근 주문 목록
- ✅ **주문 관리** - 주문 목록, 주문 상세, 타임라인 UI
- ✅ **네비게이션** - 대시보드, 주문, 고객, 영상, 설정 메뉴
